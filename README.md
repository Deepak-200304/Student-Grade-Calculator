# Project Overview
It is a Python program that takes a student’s name and marks, calculates the grade (A, B, C, D, or F), and displays an encouraging message.

The project focuses on building core programming skills such as:

-User input handling

-Conditional logic

-Functions

-Loops for validation

-Clean and readable output
